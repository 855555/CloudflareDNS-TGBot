# CloudflareDNS-TGBot


在 Linux 系统中，使用以下命令来安装 Git：

## 步骤 1: 安装 git

```bash
```bash
sudo apt update
sudo apt install git -y && sudo apt install nodejs npm
```

> 提示：这可能会花点时间，尤其是在性能不太突出的 VPS 上。

## 步骤 2: 检验安装是否成功

安装完成后，可以通过以下命令检查 Git 和 Node.js 是否已正确安装：

```bash
git --version
node -v
npm -v
```

如果这些命令返回了相应的版本号，说明安装成功。
